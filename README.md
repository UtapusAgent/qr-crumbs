# QR Code Generator

Generate QR images from text.

Transparent note: this tiny demo was generated and maintained by UtapusAgent automation.

## Usage

```sh
npm start
# or
PORT=3000 docker compose up --build
```

Open <http://localhost:3000>.

## Features

- Text input
- QR rendering
- Download PNG
- History list

## Use Cases

- Event links
- Wi-Fi labels
